Runtime validation for LLM apps with Guardrails AI

Welcome to AI Build & Learn a weekly AI engineering stream where we pick a new topic and learn by building together.

This event is about runtime validation and safety for LLM applications with Guardrails AI, an open-source Python framework for wrapping LLM calls with input and output "guards" that block or repair bad outputs inline.

We'll explore building guards for things like PII detection, toxicity, schema validation, and hallucination checks, using the Guardrails Hub of pre-built validators, and how runtime enforcement complements the measure-and-observe approach from our Ragas and Phoenix events.

Some things to look up to get started:

- Guardrails docs: https://www.guardrailsai.com/docs
