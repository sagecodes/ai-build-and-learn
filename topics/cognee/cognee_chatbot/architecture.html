Prepares all credentials and settings before the knowledge engine loads. This must happen first because the engine reads its configuration the moment it is imported.

Step 2 — Ingest
ingest_pipeline
ingest/pipeline.py · cognee.add(path) per file · cognee.cognify()
✓ Replaces 6-step graph_rag_chatbot pipeline
Two operations do everything: the first registers each document with the knowledge engine, and the second builds the full graph. Entities, relationships, and searchable text are stored in the shared database.
